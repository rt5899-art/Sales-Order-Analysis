Sales Order Analysis 📦
An end-to-end sales data analytics project built with Power BI and Excel, providing deep insights into sales orders, revenue performance, customer behavior, and product trends through interactive dashboards.

📊 Project Overview
This project analyzes sales order data to help businesses track performance, identify top-selling products, understand customer purchasing patterns, and monitor order fulfillment. The Power BI dashboard enables stakeholders to explore key sales metrics and make informed, data-driven decisions.

📁 Repository Contents

🔍 Key Analysis Areas

Sales Performance — Total revenue, order volume, and growth trends over time
Product Analysis — Top-selling products, category-wise revenue, and profit margins
Customer Insights — Customer segmentation, repeat purchase rates, and top buyers
Regional Analysis — Sales distribution across regions, cities, and territories
Order Fulfillment — Order status tracking, delivery timelines, and pending orders
Monthly & Yearly Trends — Seasonality patterns and year-over-year comparisons


🛠️ Tools & Technologies

Power BI Desktop — Data modeling, DAX measures, and interactive dashboards
Microsoft Excel — Source dataset and data preparation
DAX (Data Analysis Expressions) — Custom KPIs, calculated columns, and measures
Power Query — Data transformation and cleaning


🚀 Getting Started
Prerequisites

Power BI Desktop (free download from Microsoft)
Microsoft Excel or any compatible spreadsheet viewer

Steps

Clone the repository

bash   git clone https://github.com/rt5899-art/Sales-Order-Analysis.git
   cd Sales-Order-Analysis

Open the dataset (optional — to inspect or modify the raw data)
Open Sales Order Dataset.xlsx in Excel.
Open the dashboard
Launch Power BI Desktop and open Sales Order Analysis.pbix.
Refresh the data source (if prompted)
Go to Home → Refresh to reload data from the Excel file. If Power BI asks to locate the source file, point it to Sales Order Dataset.xlsx in the same folder.


📌 Dashboard Highlights

Overview Page — High-level KPIs: total sales, total orders, average order value, and revenue growth
Product Performance — Best and worst performing products by revenue and quantity
Customer Analysis — Top customers, new vs. returning buyers, and segment breakdown
Regional Sales Map — Geographic distribution of orders and revenue
Trend Analysis — Month-over-month and year-over-year sales trends


📂 Dataset
The dataset (Sales Order Dataset.xlsx) contains sales transaction records with fields including:

Order ID, Order Date, Ship Date
Customer Name, Customer Segment, Region
Product Name, Category, Sub-Category
Quantity Ordered, Unit Price
Total Sales, Discount, Profit
Order Status and Shipping Mode


Note: This dataset is for educational and analytical purposes only.
